# codegen-rust
Rust 实现的代码生成器

## 起步 Getting started

```bash
# help
./codegen --help

# generate some code
./codegen --databases=test-db,test-db2 --tables=test-tb,test-db2

# generate all code
./codegen --databases=* --tables=*
```
